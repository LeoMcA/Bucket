Gets a feed from its source

get_feed(mixed $url) : mixed

Gets feed(s) from the site provided to the function using the SimpleXML php extension. This function works recursively to get feeds if an array of urls is passed in.

Usage: // single feed $feed = $this->planet->get_feed('http://www.example.com/feed.rss');

    // multiple feeds
    $feeds = $this->planet->get_feed(array(
        'http://www.example.com/feed.rss',
        'http://www.anotherexample.com/feed.rss'
    ));

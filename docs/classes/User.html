Signs a user in

sign_in(string $user_email, int $user_status, string $redirect_path) 

Sets a session cookie for a user so they stay signed for a set amount of time, defined in the CodeIgniter config file application/config/config.php. The user will then be redirected to the path passed to the function

Parameters

$user_email

string

The user to sign in

$user_status

int

The status of the user (e.g. 0 = admin (by default)

$redirect_path

string

The uri to redirect the user to once complete
